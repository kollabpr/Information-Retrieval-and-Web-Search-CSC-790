Navigate to src folder and run the following commands to generate the pickle files:

1. BUILD TF, TFIDF VECTORS: python tf.py && python tfidf.py

2. BUILD WF, WFIDF VECTORS: python wf.py && python wfidf.py

3. BUILD SIMILARITY MATRICES: python similarity.py

In the main folder:
1. To get the required output execute the command: python runner.py
